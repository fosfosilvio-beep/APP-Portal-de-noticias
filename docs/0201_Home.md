# [0201] Home (Página Principal)

Responsável por exibir o feed principal de notícias, o player de transmissão ao vivo e as seções categorizadas.

## Informações Gerais
- **Rota**: `/`
- **Arquivo**: `src/app/page.tsx`
- **Responsabilidade**: Hub central de consumo de conteúdo.

## Componentes Utilizados
- `Header`: Navegação e troca de categorias stateful.
- `SmartPlayer`: Exibição da Live ou vídeo de destaque.
- `LiveChat`: Chat em tempo real (visível apenas quando `is_live` é true).
- `AutomatedNewsFeed`: Agregador de notícias externas.

## Contrato de Dados (Fetch)
- **Tabela `configuracao_portal`**: Busca status de live e banners de anúncio.
- **Tabela `noticias`**: Lista as últimas 80 notícias ordenadas por `ordem_prioridade` e `created_at`.
- **Tabela `biblioteca_lives`**: Carrega o catálogo de vídeos passados para a aba "Biblioteca".

## Lógica de Estado
- `categoriaAtiva`: Controla o que é renderizado no feed principal (Início, Polícia, Política, etc).
- `isLive`: Estado reativo via canal do Supabase (`home_config_changes`) que detecta quando a live é ligada/desligada no admin.

## Design
- Layout "Bento Grid" para os destaques principais.
- Faixas horizontais para categorias específicas (Polícia, Política).
- Barra lateral (Sidebar) com Clima, Anúncios e "Giro 24h".
