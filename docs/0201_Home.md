# [0201] Home (Página Principal)

Responsável por exibir o feed principal de notícias, o player de transmissão ao vivo e as seções categorizadas.

## Informações Gerais
- **Rota**: `/`
- **Arquivo**: `src/app/page.tsx`
- **Responsabilidade**: Hub central de consumo de conteúdo.

## Componentes Utilizados
- `Header`: Navegação e troca de categorias stateful. Suporta **Smart UI Fallback** e inclui um **CTA interativo de Podcast** que substitui o antigo indicador de Live.
- `SmartPlayer`: Exibição da Live ou vídeo de destaque com Badges dinâmicos (REC, EXCLUSIVO).
- `HeroBanner`: Carrossel dinâmico da Home com suporte a animações Slide-Up (Title, Subtitle, Categoria).
- `LiveChat`: Chat em tempo real (`youtube`/`facebook`), agora repaginado para **Dark Mode**.
- `AutomatedNewsFeed`: Agregador de notícias externas.

## Contrato de Dados (Fetch)
- **Tabela `configuracao_portal`**: Busca status de live e banners de anúncio. Emite sinais Realtime (`smart_player_config`).
- **Tabela `noticias`**: Lista as últimas 80 notícias ordenadas; NÃO envia mais dados de vídeo para o player principal (código limpo).
- **Tabela `biblioteca_webtv` e `biblioteca_lives`**: Fontes primárias de dados de vídeo quando o portal não está Ao Vivo.

## Lógica de Estado (Bifurcação Live vs Biblioteca)
- **Modo Live Actived**: Se `isLive` for `true`, o player foca na transmissão ao vivo (`url_live_facebook`), e o componente `LiveChat` (Dark Mode) é exibido lateralmente no layout principal. O carrossel inferior é oculto via transição CSS suave.
- **Modo Fallback / Biblioteca**: Se `isLive` for `false`, o player resgata automaticamente conteúdos de podcast do acervo acompanhado de um carrossel interativo sob o player.
- `configuracao_portal.ui_settings`: Central de customização (Design System JSONB). Define cor principal, logotipo (Imagem/Texto dinâmico), mensagem Marquee de `Breaking News` e chaveia a visibilidade dos Widgets.
- **Widgets da Sidebar (Premium)**:
  - **Clima**: Apresenta Sensação Térmica e Umidade baseados no design system "Deep Night".
  - **Giro 24h / Plantão Policial**: Filtragem em tempo real sobre `todasNoticias`. Renderização controlada administrativamente vía `ui_settings.widgets_visibility`.
- `categoriaAtiva`: Controla o que é renderizado no feed principal.

## Design
- Layout "Deep Space" (Dark Mode Root) estruturado utilizando o modelo "Bento Grid".
- Os containers receberam tratamento em *Glassmorphism* (Backdrops desfoques `bg-zinc-900/40`), reflexões sutis de luz (sombras coloridas baseadas na notícia).
- Layout bifurcado com transições CSS suaves (Framer/CSS).
- A interface de Live mescla organicamente com o Chat e as Sidebars, removendo totalmente o aspecto genérico branco (`bg-slate-50`).
